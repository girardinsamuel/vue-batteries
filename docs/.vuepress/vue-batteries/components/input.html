# props

  • leading-icon String (optional) default: ''

    Name of leading icon. This is using Icon component that you can configure to suit your own set of icons (or to use an external library).

  • trailing-icon String (optional) default: ''

  • password-toggle Boolean (optional) default: true

    Allow user to show input content for password type's input by clicking on the trailingIcon. Only shown if input type is password.

  • status-icon Boolean (optional) default: false

    Show status icon in trailingIcon slot when input status is defined. Icons can be defined in global input configuration object for each status.
